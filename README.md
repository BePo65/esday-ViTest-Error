## esday-ViTest-Error

> esday-ViTest-Error is a JavaScript library that tries to reproduce the error `Could not resolve "chromium-bidi/lib/cjs/bidiMapper/BidiMapper"` in the [esday library](https://github.com/g-mero/esday).
